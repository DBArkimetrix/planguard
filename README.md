# Agent Framework v3

A production-oriented framework for AI-agent development with:

- plan-first execution
- phased backlog and sprint planning
- safety gating before implementation
- orchestration and dependency sequencing
- regression and collision review
- CI guardrails
- automation scripts for validation and scoring

## Required workflow

PLAN → BACKLOG → SPRINT PLAN → SAFETY CHECK → ORCHESTRATE → IMPLEMENT → UPDATE DOCS → HANDOFF
